Design Inspiration

The Nexus Crux logo draws from multiple design influences:

  • Network Topology Diagrams - Hub-and-spoke architecture visualization
  • Molecular Structures - Interconnected systems with central bonding
  • Stripe's Minimal Aesthetic - Clean, purposeful design without decoration
  • Federated Architecture - Distributed yet connected nodes around a core
  • Data Visualization - Clear representation of relationships and flows
  • Geometric Precision - Hexagonal symmetry and mathematical balance

Usage Guidelines

Do's ✓

  • Maintain clear space equal to one orbiting node radius (≈9% of logo size)
  • Use minimum size of 32×32px for icons (digital)
  • Use inverse version on dark backgrounds (#0A1A2F or darker)
  • Keep aspect ratio when scaling - network structure must remain proportional
  • Use SVG format for perfect scalability
  • Choose appropriate variant for context (color vs. monochrome)
  • Ensure network structure is clearly visible at display size

Don'ts ✗

  • Don't change logo colors (Teal core + Lime nodes) except for approved variants
  • Don't rotate, skew, or distort the network structure
  • Don't add effects (shadows, outlines, glows, extra gradients)
  • Don't alter spacing between core and orbiting nodes
  • Don't change the number of nodes (must be 6 in hexagonal pattern)
  • Don't stretch or compress - breaks the geometric precision
  • Don't place on busy backgrounds that obscure the network
  • Don't remove or hide connection lines - they're essential to "Nexus"

Technical Specifications

Network Structure (100×100 viewBox)

  • Core Node: Center (50, 50), radius 18 units
  • Orbiting Nodes: 6 nodes, radius 9 units each
  • Node Positions: Top (50,10), Top-right (84.64,25), Bottom-right (84.64,75), Bottom (50,90), Bottom-left (15.36,75), Top-left (15.36,25)
  • Connection Lines: 6 lines from core to each node, 2 units stroke width, 60% opacity
  • Layout: Hexagonal/symmetric arrangement for visual balance

Color Application

  • Primary Variant: Teal core (#2AD1C8) + Lime nodes (#A6F750) + Teal lines
  • Inverse Variant: Full opacity for better visibility on dark backgrounds
  • Monochrome Variant: Navy (#0A1A2F) for all elements, lines at 60% opacity
