1. **Click**: "+ Add product"
2. **Fill in**:
   - Name
   - Description
3. **Add First Price**:
   - Amount (e.g., 399.00)
   - Currency: GBP
   - Billing: Monthly
   - Recurring: ‚úÖ Yes
4. **Add Second Price**:
   - Amount (e.g., 3830.00)
   - Currency: GBP
   - Billing: Yearly
   - Recurring: ‚úÖ Yes
5. **Save product**
6. **Copy both Price IDs**
